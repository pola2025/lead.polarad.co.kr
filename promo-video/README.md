# POLA Lead System 프로모션 영상

## 영상 링크

- **YouTube**: https://youtu.be/C1k-NlmqO3k

## 프로젝트 정보

- **제작 도구**: Motion Canvas
- **해상도**: 4K (3840x2160)
- **길이**: 약 45초

## 씬 구성

1. `opening` - 오프닝 타이틀
2. `adToLanding` - 광고 → 랜딩 페이지 전환
3. `customerFlow` - 고객 플로우 (카카오 로그인 → 신청완료)
4. `realtimeAlert` - 실시간 텔레그램 알림
5. `portalLeads` - 포털 리드 관리
6. `portalStats` - 포털 통계 대시보드
7. `features` - 주요 기능 소개
8. `closing` - CTA 클로징

## 실행 방법

```bash
cd promo-video
npm install
npm run dev
```

브라우저에서 `http://localhost:9000` 접속

## 렌더링

1. Motion Canvas UI에서 **Rendering** 탭 클릭
2. **Output** → **Video** 선택
3. **Render** 클릭
