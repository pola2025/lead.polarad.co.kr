카카오 로그인 - 로컬 개발 환경 설정

localhost:3000에서 카카오 로그인 테스트를 위한 설정 가이드

카카오 개발자 콘솔 바로가기:

https://developers.kakao.com/console/app/1373602

📋 설정 체크리스트

1

카카오 로그인 활성화

카카오 로그인 기능을 활성화합니다.

📍 경로:

제품 설정 → 카카오 로그인 → 일반 → 사용 설정

✅ 상태를 ON으로 설정

2

동의항목 설정 (이메일)

사용자 이메일을 수집하기 위한 동의항목을 설정합니다.

📍 경로:

제품 설정 → 카카오 로그인 → 동의항목

카카오계정(이메일) 항목의 [설정] 버튼 클릭
→ "필수 동의" 또는 "선택 동의" 선택
→ "동의 목적" 입력 (예: 회원가입 및 본인확인)

3

리다이렉트 URI 추가

카카오 로그인 후 돌아올 콜백 URL을 등록합니다.

📍 경로:

제품 설정 → 카카오 로그인 → 고급 → 리다이렉트 URI

추가할 URI (복사해서 사용):

⚠️ 프로덕션 URI도 함께 등록해야 합니다:
https://lead.polarad.co.kr/api/auth/kakao/callback

4

JS SDK 도메인 추가

JavaScript SDK를 사용할 도메인을 등록합니다.

📍 경로:

앱 설정 → 플랫폼 키 → JavaScript 키 → [JS SDK 도메인] 클릭

추가할 도메인 (복사해서 사용):

⚠️ 프로덕션 도메인도 함께 등록해야 합니다:
https://lead.polarad.co.kr

🔧 자주 발생하는 에러

KOE006

리다이렉트 URI가 일치하지 않음 → Step 3 확인

KOE101

카카오 로그인이 비활성화됨 → Step 1 확인

KOE004

앱 키가 올바르지 않음 → 환경변수 확인

✅ 설정 완료 후 테스트

  1. 개발서버 시작: npm run dev
  2. 브라우저에서 랜딩 페이지 접속: http://localhost:3000/l/polamkt
  3. "카카오로 시작하기" 버튼 클릭
  4. 카카오 로그인 진행
  5. 이메일이 자동으로 폼에 채워지는지 확인
